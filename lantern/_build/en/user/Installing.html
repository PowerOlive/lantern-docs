Installing and Uninstalling¶

This section provides details on how to install and uninstall Lantern on multiple operating systems.

Once you are invited, you will receive an email with links to download the installer for your operating system.

Linux¶

Supported Versions¶

Currently, installation on Linux is only available on Ubuntu. We greatly appreciate any contributions for other Linux distributions or help with improvements to make better use of package managers. You can also build from source.

Installing¶

The invitation to Lantern has a link to an installation script that:

  • Downloads a .deb file.
  • Creates the uninstall script in the /opt/lantern-net-installer directory.
  • Creates a user preferences directory.
  • Does not auto-launch a Lantern service.
  1. Click Ubuntu 12.04+ in the invitation email.

    A browser window opens and dialog appears asking you to open or save the installer.

  2. Open a new terminal window and change to the directory where you downloaded the installer.
    cd Downloads/
  3. Make the installation script executable.
    chmod 700 lantern-net-installer_unix.sh
  4. Run the installation script.
    sudo ./lantern-net-installer_unix.sh

    The terminal window displays a message indicating the installer is starting and the Lantern Fetcher dialog appears.

    Note

    There is no status message indicating the installation is complete.

    If you receive a message indicating there is no suitable Java Virtual Machine, you need to download Java. See the Ubuntu documentation for details.

  1. Click the Ubuntu button and type Lantern.

  2. Click the Lantern icon.

    The Lantern Setup window appears.

    Note

    If you receive a message about Google Chrome not being installed, you need to download it.

    See https://www.google.com/intl/en/chrome/browser/ for details.

  3. See Setting Up Lantern.

Uninstalling¶

  1. In a terminal window, change to the installation directory.
    cd /usr/local/lantern-net-installer/
  2. Run the uninstall program.
    sudo ./uninstall

    The Lantern Uninstall window appears.

  3. Click Next to start uninstalling lantern.

    A message appears when the uninstallation is complete.

  4. Click Finish.

Mac¶

Supported Versions¶

Lantern supports OS/X version 10.6 and higher, 64-bit only.

Note

During installation, Lantern checks if you have Java installed. If not, it will download the Java Runtime Environment from the web.

Installing¶

  1. Click the Mac OS X 10.6+ link in the invitation email.

  2. Double-click the .dmg file that downloads, then double-click on the Lantern Installer icon.

    The Lantern Fetcher window appears and starts downloading the most recent installation files.

    Important

    As of beta2 the installer will not work if you are not running as an admin user. If you are not the admin user, nothing will happen when you click the installer icon. We are working on getting it to prompt for the admin password instead of not responding.

  3. In the password dialog that appears, enter your administrator user name and password. This allows Lantern to install and make changes to your proxy settings so that your internet connection can be shared.

    The installer downloads and extracts the files. When the installation is complete, the Lantern window appears.

  4. See Setting Up Lantern.

Uninstalling¶

  1. Right-click on the Lantern status bar icon and select Quit Lantern.

  2. In the Applications folder, move Lantern.app to the Trash.

    This also uninstalls Lantern’s Java Runtime Environment.

  3. To delete all your settings, open a terminal window and change to the directory in your Home directory.
    cd ~

    Note

    If you don’t delete your settings, Lantern uses them the next time you install the application.

  4. Delete the ./lantern directory.
    rm -r .lantern

Windows¶

Supported Versions¶

Lantern supports Windows version XP and higher and requires that Google Chrome is installed. If you don’t have Chrome you can get it here.

Installing¶

  1. Click the Windows XP+ link in the invitation email.

    Warning

    It is important to follow the next steps to make sure that the digital signature is valid.

  2. Right click the .exe file that downloaded and click the Digital Signatures tab.

  3. In the General tab, click View Certificate.

  4. Under Details, verify the signature.

    Each .exe file is digitally by signed by Brave New Software Project, Inc. For June 11, 2013 to June 11, 2016, the SHA1 thumbprint should be ‎44:99:0c:72:fb:a0:e6:dd:f9:2a:b9:34:46:02:e7:56:00:d8:be:3c.

  5. Once you have verified the signature, double-click the .exe file. If you are prompted to allow the installer to make changes to your computer, click Yes.

    When installation is complete, the Lantern window appears.

  6. See Setting Up Lantern.

Uninstalling¶

  1. Open the Control Panel and click Programs.

  2. In the list of installed programs, click the Lantern.

  3. Click the Uninstall/Change button.
  4. Click Next.

    Lantern is uninstalled.

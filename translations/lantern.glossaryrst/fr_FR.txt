Lexique
========

.. lexique::

HTTP
	Hypertext Transfer Protocol. Protocole permettant de transférer divers contenus comme des pages web
	sur internet. Ce contenu est envoyé en clair et de ce fait, n'importe quel
	`intermediaire <#wiki-intermediary>`__ est potentiellement capable de le lire pendant son
	transfert. HTTP n'est donc pas sécurisé. `HTTPS <#wiki-https>`__ est la
	version sécurisée d'HTTP.

HTTPS
	Version sécurisée d'`HTTP <#wiki-http>`__. L'information est envoyée cryptée pour
	empécher les `intermediaires <#wiki-intermediary>`__ de la lire
	pendant le transfert.

Install4g 
	Install4g est un programme que nous utilisons pour faire en sorte que l'installation de Lantern soit plus simple pour toutes les plateformes.
	
Intermédiaire
	An entity on a computer network in between the requester of some content
	and the server which hosts it, e.g. a `proxy <#wiki-proxy>`__ or an
	`fournisseur d'accès internet <#wiki-isp>`__.

FAI
	Fournisseur d'Accès Internet, organisation qui fournit un accès à internet
	access. Traffic leaving a computer's local network (in a home, office,
	internet cafe, etc.) goes here before either being forwarded to its
	destination, or blocked in the case that the ISP employs censorship.
	Traffic bound for your computer likewise must pass through your ISP
	before it can reach you.

Pair
	Another user you're connected to on a computer network.

Peer-to-Peer (P2P)
	Describes a network architecture where each computer in the network can
	act as client and server, as opposed to just one or a few servers
	providing service to many clients. Peer-to-peer networks are
	decentralized and therefore more difficult to shut down.

Relai
	An `intermediary <#wiki-intermediary>`__ which forwards traffic between
	a user requesting content and the server that hosts it, often because
	the user cannot reach the server directly due to censorship.

 Proxy Système
	A program which proxies traffic for all programs on a computer which
	accède au réseau.

Установка и удаление
===========================
В этом разделе приведена информация по установке и удалению Lantern для разных операционных систем.

После того, как вас пригласят, вы получите сообщение по электронной почте, в котором будет ссылка на программу установки для вашей операционной системы.

.. image:: https://github-camo.global.ssl.fastly.net/6a8e1af9a0053337263343727132eacab1920d88/687474703a2f2f692e696d6775722e636f6d2f347652615436382e706e67
	:alt: Приглашение Lantern 

Linux
-----

Поддерживаемые версии
^^^^^^^^^^^^^^^^^^
В настоящее время установка на Linux возможна только для Ubuntu.
Мы весьма ценим любое `содействие <../../developer/html/contributing.html>`_  в разработке версий программы для других дистрибутивов Linux и помощь в улучшении использования менеджеров пакетов. Вы также можете сами `скомпилировать программу из следующего источника  <https://github.com/getlantern/lantern/blob/master/README.md#wiki-setting-up-a-development-environment>`_.

Установка
^^^^^^^^^^ 
Письмо с приглашением в Lantern содержит ссылку на установочный скрипт, который: 

* Скачивает `.deb` файл. 
* Создает скрипт деинсталяции программы в каталоге `/opt/lantern-net-installer` .
* Создает каталог с настройками пользователя.
* Запрещает автозапуск сервиса Lantern.

#. Нажмите на ссылку **Ubuntu 12.04+**, указанную в письме-приглашении.
	Откроется окно браузера и появится диалоговое окно с вопросом, хотите ли вы открыть или сохранить файл установки. 

	.. image:: http://i.imgur.com/justLyz.png
		:alt: Диалог открытия файла

#. Откройте окно терминала и измените текущий каталог на тот, в который вы сохранили файл установки.
	:: 
		
		cd Downloads/

#. Измените права доступа для скрипта установки, чтобы дать ему права на исполнение.
	::
	
		chmod 700 lantern-net-installer_unix.sh

#. Запустите скрипт установки.
	::
		
		sudo ./lantern-net-installer_unix.sh
	
	Окно терминала отобразит сообщение о том, что установщик начал свою работу, и появится диалог сборщика Lantern.
	
	.. image:: http://i.imgur.com/S2hBiEY.png
		:alt: Окно терминала и диалог сборщика Lantern с указанием статуса загрузки.
		
	.. примечание: Сообщения о том, что установка завершилась, отображаться не будет. 
		
		В случае, если вы получите сообщение о том, что не найдена подходящая версия виртуальной машины Java, вам нужно будет скачать Java. Подробности смотрите в `Документации по Ubuntu  <https://help.ubuntu.com/community/Java>`_ .

5. Нажмите на кнопку **Ubuntu** и введите Lantern.

6. Нажмите на значок **Lantern**
	Откроется окно настройки Lantern.
	
	.. image:: http://i.imgur.com/nnXoFjr.png
		:alt: Запрос в окне настройки о выборе режима работы Lantern: "Дать доступ" или "Получить доступ".
	
	.. примечание:: Если вы получите сообщение о том, что у вас не установлен Google Chrome, вам нужно будет его скачать.
		
		Подробности по ссылке: https://www.google.com/intl/en/chrome/browser/ 

#. См. :ref:`setting-up-lantern`.

.. _uninstalling-linux:
Удаление
^^^^^^^^^^^^

1. В окне терминала перейдите в каталог установки.
	::
	
		cd /usr/local/lantern-net-installer/
		
2. Запустите программу деинсталляции
	::
	
		sudo ./uninstall
	
	Появится окно деинсталляции Lantern.
		
	.. image:: http://i.imgur.com/PW3hiCF.png
		:alt: Окно терминала с командой удаления Lantern.
		
3. Нажмите **Далее** для запуска процесса удаления Lantern.
	Когда удаление завершиться, появится соответствующее сообщение.
	
	.. image:: http://i.imgur.com/7XZayjD.png
		:alt: Окно деинсталляции Lantern с сообщением о том, что Lantern был успешно удален.
	
4. Нажмите **Завершить**.

.. _installing-mac:
Mac
---
Поддерживаемые версии
^^^^^^^^^^^^^^^^^^

Lantern поддерживает версии Mac OS/X начиная с 10.6 и выше,
только 64-битные версии. 

.. note:: В процессе установки, Lantern проверит установлен ли пакет Java. Если нет, то установщик скачает Java Runtime Environment из сети.

Установка
^^^^^^^^^^

#. В письме-приглашении нажмите на ссылку **Mac OS X 10.6+**.

#. Двойным щелчком запустите файл **.dmg**, который скачается, затем двойным щелчком по ярлыку запустите **Lantern Installer*.
	Появится окно сборщика Lantern и начнется скачивание установочных файлов последней версии.
	
	.. image:: https://www.evernote.com/shard/s209/sh/b39a5f0d-4aa9-4518-8a26-fd0a86de8737/9d3090f5eab319830f4510bc13bba90e/deep/0/Lantern%20Fetcher.png
		:alt: Загрузчик Lantern.
		
	.. Важно:: В случае если у вас *не* будет работать установщик версии beta2, значит вы зашли в систему не в качестве администратора. Когда вы щелкните по значку установщика - ничего не произойдет. Мы `работаем <https://github.com/getlantern/lantern/issues/819>`_  над тем, чтобы установщик спрашивал пароль Администратора вместо того, чтобы не реагировать вовсе.

#. В диалоговом окне с запросом пароля введите имя пользователя-администратора и его пароль. Это позволит продолжить установку Lantern и внести изменения в настройки прокси с тем, чтобы ваше интернет соединение могло быть общим.
	.. image:: https://www.evernote.com/shard/s209/sh/c92df454-472a-4ce1-a578-fc06843802d7/04fd633f4deb9fbeecc44562dff2ea78/deep/0/Screenshot%208/16/13%207:52%20PM.png
		:alt: Окно с запросом имени пользователя и пароля для разрешения внесения изменений.
	
	Установщик скачает и распакует файлы. Когда установка завершится, появится окно Lantern.

	.. image:: https://dl.dropboxusercontent.com/u/253631/Installing_Mac_Setup_Window.png
		:alt: Появляется окно программы Lantern с предложением начать процесс настройки.
		
#. См. :ref:`setting-up-lantern`.
	
Удаление
^^^^^^^^^^^^

1. Щелкните правой кнопкой на иконке Lantern в строке состояния и нажмите **Выход из Lantern**.

2. В папке **Приложения** переместите **Lantern.app** в **Корзину**.
	Это действие также удалит и Java Runtime Environment.

3. Для удаления ваших настроек, откройте окно терминала и перейдите в вашу **Домашнюю** папку.
	::
	
		cd ~
		
	.. примечание:: Если вы не удалите свои настройки, Lantern использует их при следующей установке.
		
4. Удалите папку **./lantern**.
	::
	
		rm -r .lantern


.. _installing-windows:
Windows
-------

Поддерживаемые версии 
^^^^^^^^^^^^^^^^^^

Lantern поддерживает версии Windows начиная с XP и требует наличие установленного Google Chrome. Если у вас нет
Chrome, вы можете скачать его `здесь: <https://www.google.com/intl/en/chrome/browser/>`_.

Установка
^^^^^^^^^^

1. В письме-приглашении нажмите на ссылку **Windows XP+**.
	.. внимание:: Очень важно следовать инструкциям и убедиться, что `цифровая подпись <http://en.wikipedia.org/wiki/Digital_signature>`_ верна.

#. Щелкните правой кнопкой на скачанном файле  **.exe** и нажмите на **Цифровые подписи**. 

#. В закладке **Общие** нажмите **Просмотреть Сертификат**. 

#. В разделе **Детали** проверьте подпись.
	Каждый .exe файл подписан цифровой подписью компании Brave New Software Project, Inc. С 11 июня 2013 года по 11 июня 2016 года, хэш SHA1 должен соответствовать 44:99:0c:72:fb:a0:e6:dd:f9:2a:b9:34:46:02:e7:56:00:d8:be:3c.

	.. image:: http://i.imgur.com/iTljBxR.jpg
		:alt: Окно информации о сертификате.
	
#. Если вы проверили подпись, дважды щелкните по **.exe** файлу. Если у вас спросят, разрешаете ли вы установщику внести изменения в ваш компьютер, нажмите **Да**.
	По завершении установки появится окно Lantern.

	.. image:: http://i.imgur.com/K6yBha5.png)
		:alt: Появляется окно программы Lantern с предложением начать процесс настройки.

#. См. :ref:`setting-up-lantern`.


Удаление
^^^^^^^^^^^^

1. В **Панели управления** откройте **Программы**.
	.. image:: http://i.imgur.com/lF9ldIE.png
		:alt: Панель управления Windows.

#.  В списке установленных программ нажмите на **Lantern**.
	
#. Нажмите на кнопку **Удалить/Изменить**.
	.. image:: http://i.imgur.com/0fTDSYF.png
		:alt: Удаление Lantern

#.  Нажмите **Далее**.
	Lantern удален.

	.. image:: http://i.imgur.com/TGQeclS.png
		:alt: Окно деинсталляции

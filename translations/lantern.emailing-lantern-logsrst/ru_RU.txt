Отправка логов Lantern по электронной почте
====================

Команда разработчиков Lantern периодически будет просить вас отправить по электронной 
почте ваши логи, чтобы лучше диагностировать выявленную неполадку.

Для этого вам нужно будет найти папку с логами на вашем компьютере:

В Windows Vista/7/8
'''''''''''''''''

`C:\Users\ИМЯ_ПОЛЬЗОВАТЕЛЯ\AppData\Roaming\Lantern\logs``

В Windows XP
''''''''''

``C:\Documents and Settings\ИМЯ_ПОЛЬЗОВАТЕЛЯ\Application Data\Lantern\Logs``

В Mac OS X
'''

``~/Library/Logs/Lantern``

В Ubuntu
''''''

``~/.lantern/logs``

Эта папка должна содержать файлы с именем типа ``java.log`,
``installer.log`` и другие. Заархивируйте всю папку в один архив.
Иногда эта папка будет содержать только файл java.log, он то и является наиболее 
важным. Вы можете его заархивировать для уменьшения размера, либо просто отправьте его нам как есть.

Приложите архив либо файл java.log к письму и отправьте на адрес contact [at]
getlantern.org. Если беспокоитесь о безопасности и хотите отправить письмо на адрес с поддержкой PGP,
сообщите нам об этом и мы предоставим вам соответствующий адрес.
